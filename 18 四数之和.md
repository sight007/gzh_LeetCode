## 18、四数之和

### 题目：

给定一个包含`n`个整数的数组`nums`和一个目标值`target`，判断`nums`中是否则在四个元素`a`、`b`、`c`、`d`，使得`a+b+c+d`的值与`target`相等？找出所有满足条件，且不重复的四元组。

### 方法一：

~~~~~~~~~~
"""
双循环，双指针碰撞


index1:[0，len-1-3]

index2:[index1+1, len-1-2]

left=index2+1
right=len-1
"""

nums=[-3,-2,-1,0,0,1,2,3]

def fourSum(nums, target):
    if len(nums) < 4:
        return []
    nums.sort()
    result = []
    for index1 in range(len(nums) - 3):
        for index2 in range(index1 + 1, len(nums) - 2):
            if nums[index2] == nums[index2 - 1]: continue
            left = index2 + 1
            right = len(nums) - 1
            while (left < right):
                sum = nums[index1] + nums[index2] + nums[left] + nums[right]
                if sum == target:
                    temp = []
                    temp.append(nums[index1])
                    temp.append(nums[index2])
                    temp.append(nums[left])
                    temp.append(nums[right])
                    result.append(temp)
                    left += 1
                    right -= 1
                elif sum < target:
                    left += 1
                    while left < right and nums[left] == nums[left - 1]: left += 1
                elif sum > target:
                    right -= 1
                    while left < right and nums[right] == nums[right + 1]: right -= 1
    return result


print(fourSum(nums, 0))
