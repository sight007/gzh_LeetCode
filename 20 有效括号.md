## 20、有效的括号

### 题目：

给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，判断字符串是否有效。

有效字符串需满足：

- 左括号必须用相同类型的右括号闭合。
- 左括号必须以正确的顺序闭合。

**注意空字符串可被认为是有效字符串。**

### 解法：

- 从左往右，一个一个填进`stack`（先进后出）
- 如果右括号有相对应的左括号，弹出右括号
- 如果字符串的长度不是偶数，直接退出

~~~python
def isValid(s):

     dic = {'{': '}',  '[': ']', '(': ')', '#':'#'}
     stack = ['#']
     
     for c in s:
        if c in dic:
            stack.append(c)
         elif dic[stack.pop()] != c:
            return False
      
    return len(stack) == 1
~~~



**难点**：

- 如果`stack`为`null`，直接`stack.pop()`的时候，会报错
- 设置原始的`stack = ['#']`，那么如果`s`中含有`#`，也会报错，因此，在字典中添加`'#':'#'`
